Жанры книг

Виды книг:

По своему социальному назначению книги делятся на учебные, научно-популярные, научные, справочные, художественные. По роду содержания их можно разделить на исторические, математические, биологические, философские, искусствоведческие, универсальные и многоотраслевые. По читательскому адресу книги делятся на две группы – для взрослого читателя и для детей.

Примеры: Романы Фэнтези Детективы
